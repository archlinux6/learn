### 1.Markdown 代码语法
可以用 ``` 包裹一段代码，并指定一种语言（也可以不指定）：

    ```javascript
    $(document).ready(function () {
    alert('RUNOOB');
    });
    ```
段落上的一个函数或片段的代码可以这样：
\`printf()\` 函数

代码区块使用 4 个空格或者一个制表符（Tab 键）。
